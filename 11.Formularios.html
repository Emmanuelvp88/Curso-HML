Veremos que son los Formularios

Los Formularios son un tipo de etiquetas las cuales tambien tienen Metodos uno de ellos es el "< input type = "" name = "" > lo cual tienen atributos que vienen siendo el tipo de entrada que se va realizar EJEMPLO: TIPO: texto,passwor,numero,email,color,rango,date,time,button. estos metodos son entradas de datos que le pedimos a el Usuario, seria parecido como a un JTextField de lenguaje Java se declara como una variable en java, se le pone el typo de dato y un nombre. en la pagina de muentran como un campo tipo "Rectangulo" donde el usuario tiene que ingresar ciertos datos y el programa tiene que tomar esos datos procesarlos y regresar una respuesta o simplemente quedarse con el dato y almacenarlo en una base de datos, siguiendo el protocolo estandar Input, procesamiento, Output.

Lo mas importante esque estos Metodos input pueden ser de diferentes tipos de datos por ejemplo: Texto, Password, Number, email, Color, range, date, Time, Button, Submit como se menciono anterior mente Tienene un nombre,atributos y propiedades como lo es "requiered" al igula que a una varible, todo lo que ingresemos en un input se guardara en value="" todos los input tienen un value ..

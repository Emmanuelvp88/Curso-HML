En esta clase veremos como insertar una Imagen, un Audio y un Video desde una ruta en nuestro ordenador "LOCAL".


Insertando una imagen

Veremos la etiqueta < img src = "" > que es para insertar una imagen.
La etiqueta "img" es para poner una IMAGENy no necesita ser cerrada, en comparacion con las etiquetas para poner un video o un audio


Insertando imagen con diferente resolucion

Misma imagen pero con los atributos "width" y "heigth"
Estos atributos de la etiqueta "img" sirven para modificar el archivo en su ancho y alto

Cabe mencionar que el protocolo es mismo para buscar elementos dentro de una carpeta EJEMPLO: "../" para una carpeta atras de donde se encuentra nuestro archivo HTML y "nombre carpeta/nombre imagen.png"para la imagen dentro de una carpeta, o rutas completas para hasta llegar a ala carpeta donde se encuentre nuestra imagen.


EJEMPLO: C:/user/flako/Escritorio/Descargas/HTML/imagen.png

Insertando un Video

En la Etiqueta: "< video src = "" >< /video" tambien es aplicable los Atributos: "width" y "height" para cambiar acho y alto del archivo y para agregar los controles por default con el atributo controls = ""

Insertando un Audio

En esta Etiqueta: < audio src = "" >< /audio > tenemos que agregar el atributo control = "" ya que sin ese atributo en la etiqueta no podremos visualizar la interfaz de reproduccion.

Listo, con esto concluimos con las etiquetas para colocar una IMAGEN, VIDEO Y AUDIO y sus atributos "Width" Y "Heigth"
